# GitHub Terminal Command Center

Cette mini application web fournit un guide pratique des commandes Git et GitHub CLI avec boutons de copie.

## Ouvrir la page

1. Ouvre le fichier `index.html` dans ton navigateur.
2. Utilise la barre de recherche pour filtrer les commandes.
3. Clique sur `Copier` pour copier une commande, ou `Copier les commandes visibles` pour tout copier d'un coup.

## Contenu

- Configuration Git
- Creation/clonage de depot
- Status, diff, log
- Ajout, commit, amend
- Branches
- Push, pull, fetch, remote
- Merge, rebase, cherry-pick
- Conflits
- Stash
- Undo, reset, revert, reflog
- Tags et releases
- Maintenance et nettoyage
- Worktrees et submodules
- GitHub CLI (`gh`)
- Alias utiles
